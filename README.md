# AI Task Manager

## 📌 Project Description
This is a full-stack AI-based task management system built using React and Node.js.  
It automatically assigns priority to tasks based on keywords.

## 🚀 Features
- Add tasks
- Automatic priority detection (High, Medium, Low)
- Backend API integration
- Real-time task display

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js (Express)
- API: REST API

## ▶️ How to Run

### Backend
cd backend
npm install
npm run serve

### Frontend
cd frontend
npm install
npm run dev

## 🌐 URLs
- Frontend: http://localhost:5173
- Backend: http://localhost:5000/tasks
